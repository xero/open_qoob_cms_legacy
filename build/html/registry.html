Registry Core Class¶

class registry¶

global registry class passing a string name to the static register function will first check the previous existance of a class. if found it’s instance is returned. otherwise, it will check if the class exists in the core classes folder, the core utilities folder, or the application controllers folder. if found, an instance of the class will be created, added to the object registry, then returned. if the db boolean is passed the database singleton is returned.
