AntiSpam Utility¶

class antispam¶

antispam class functions for using the Akismet spam protection system. check out the full api docs at: http://akismet.com/development/api/

$vars = array(
    'user_ip'               => $_SERVER['REMOTE_ADDR'],
    'user_agent'            => $_SERVER['HTTP_USER_AGENT'],
    'referrer'              => $_SERVER['HTTP_REFERER'],
    'comment_author'        => $name,
    'comment_author_email'  => $from,
    'comment_content'       => $msg
);
if($this->antispam->test($vars)) {
  //---spam!
  header("Location: ".QOOB_DOAMIN."spam");
} else {
  //---send email
  mail($to, $subject, $msg, $from_header);
  header("Location: ".QOOB_DOAMIN."thank_you");
}
